git / github
git : 버전 관리, 백업, 협업
github : 서비스

작업 트리 : 폴더, 실제 문서 작업하는 공간
스테이지 : 버전 만들기 전, 버전으로 만들 파일이 대기하는 공간
저장소 : 스테이지에서 대기하던 파일들을 버전으로 만들어 저장하는 곳, 리포 -> 사용자들에게 보여지는 곳

git add
git commit : 버전으로 만들기

branch

mkdir 폴더명 : 폴더 생성
cd 폴더명 : 해당 폴더 이동
git init

git status : 폴더 안 문서 상태 보기
git add . : 스테이지 안 모든문서

git restore 파일명 : 스테이지 전 수정한 파일을 되돌리기
git reset 파일명 : 스테이지에서 내리기

git commit -am ""    : 커밋
- : 옵션
a : all
m : message

git reset HEAD^ : 제일 최근 커밋을 이전 것으로 돌리기

git log : 커밋한 것 보여주기
